# Writing Voice

Use this reference for writing, rewriting, prose, posts, letters, and personal expression.

## Determine Transformation Freedom

- `exact`: preserve content-bearing words and qualifiers when possible; change only what is needed for correctness or the requested operation;
- `restrained`: improve clarity and organization while preserving propositions, intensity, and recognizable voice;
- `open`: create freely within the requested intent and factual boundaries.

The current message determines the mode. An explicit request to preserve, keep, only adjust, or not change enters exact mode.

## Proposition Lock

Extract the source propositions before drafting. Preserve unless the user explicitly permits reinterpretation:

- trend versus frequency;
- negation, modality, uncertainty, hesitation, and degree;
- agency and causal direction;
- chronology;
- emotional intensity;
- numbers, ranges, and scope;
- stated facts versus suggested possibilities.

For exact and restrained work, compare source and candidate clause by clause for additions, removals, strengthening, weakening, and reassignment to a different clause.

Near-synonyms may change meaning. Wanting, willingness, ability, obligation, intention, and action are not interchangeable.

## Candidate Selection

Form a minimum-change candidate first. Form a more interpretive candidate only when the requested transformation permits it. Select by:

1. explicit writing request;
2. proposition preservation;
3. user voice and established preference;
4. clarity;
5. minimum unrequested transformation.

Reject a smoother or more vivid candidate when it adds a person, action, motive, habit, cause, body state, relationship, event, or outcome unsupported by the source.

Returning the original unchanged is valid when it already satisfies an exact request. Visible editing effort is not a quality target.

## Requested Style

When the user requests literary, public-facing, intimate, restrained, or another texture, treat it as permission to explore candidates, not as a fixed recipe. Let the source and user examples determine the transformation. Do not impose a reusable rhythm, image type, sentence shape, or ending.

For A/B preference, record why the user preferred one version. Generalize the decision reason and do not copy the winning sentence into generation guidance.

## Quality Gate

A valid rewrite lets the user compare source and output without finding an unpermitted new fact, changed proposition, or lost qualifier.
