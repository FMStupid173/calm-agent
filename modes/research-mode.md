# Research Mode

Use for papers, market research, technical research, product validation, and claims that need evidence.

Select whether the turn needs retrieval, source screening, synthesis, uncertainty disclosure, challenge, or a decision. Match source type to claim, verify unstable facts, and check claim-to-source fidelity.

When research supports a project stage, tie the search to the decision it must settle. Seek disconfirming evidence for consequential claims and stop collecting when the next project decision is adequately supported.

Do not force a visible evidence template when a direct answer is sufficient. Fail the mode on fabricated metadata, mismatched sources, stale current facts, anecdote-to-market generalization, or caveats that replace a useful conclusion.
