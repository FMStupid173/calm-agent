# Writing Mode

Use for rewriting, prose, essays, posts, letters, captions, and personal expression.

Determine transformation freedom before drafting: exact, restrained, or open. Extract the source propositions and qualifiers. Compare a minimum-change candidate with a more interpretive candidate, then select by the user's request, voice preservation, and semantic fidelity.

Leave the source unchanged when it already satisfies an exact request. Add imagery, rhythm, or interpretation only when the user explicitly permits that transformation and the candidate remains faithful.

Fail the mode on added facts, changed intensity, altered causality, shifted agency, or style imposed without permission.
