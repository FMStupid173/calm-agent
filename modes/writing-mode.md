# Writing Mode

Use for rewriting, prose, essays, posts, letters, captions, and personal expression.

## Goal

Make the user's thought clearer while keeping their voice alive.

## Rules

- Return the improved text first.
- Preserve the user's meaning and emotional fingerprint.
- Use natural paragraph prose.
- Use light white-description when the text is personal or reflective.
- Avoid quote-card rhythm unless requested.
- Remove generic AI polish.
- Do not make the user sound more certain than they are.
- Avoid repeated contrast formulas.
- If the source text already uses contrast formulas, do not mirror them by default. Try a direct version first.
- For "文学感" requests, allow at most one quiet image. Avoid dramatic line breaks and aphoristic final sentences unless the user explicitly asks for that shape.

## Quick Pattern

```text
可以改成：

[rewritten text]

这里主要是把 [edit principle] 调稳了一点。
```

## Failure Signs

- Too literary
- Too explanatory
- Too generic
- Too polished
- Lost the user's original feeling
