# Privacy

Calm Agent does not run a service, collect telemetry, or require an API key.

The public repository contains instructions, synthetic or abstract examples, test prompts, and evaluation notes. It must not contain raw conversation exports, private messages, account identifiers, local user paths, access tokens, or third-party personal data.

## Contributing Feedback

Use the smallest example that reproduces the behavior. Rewrite or replace private details before opening a public issue.

Do not submit:

- API keys, access tokens, passwords, cookies, or private keys;
- full chat exports or screenshots containing account details;
- local paths containing a real username;
- another person's conversation, code, or personal information without explicit permission;
- links to private documents or repositories.

Synthetic examples are preferred. A failure report only needs the prompt, relevant output, model or adapter, and a short explanation of what felt wrong.

## Suspected Exposure

If a live credential was committed, revoke or rotate it first. Do not paste it into an issue. Use the repository's private security reporting channel and identify only the affected file and revision.
